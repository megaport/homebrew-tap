# Megaport Homebrew Tap

Official [Homebrew](https://brew.sh/) tap for [Megaport CLI](https://github.com/megaport/megaport-cli).

## Installation

```bash
brew tap megaport/tap
brew install megaport-cli
```

## Upgrading

```bash
brew update
brew upgrade megaport-cli
```

## Uninstalling

```bash
brew uninstall megaport-cli
brew untap megaport/tap
```
